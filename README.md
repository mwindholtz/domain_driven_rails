# Domain Driven Rails

## Goals

* Provide generators to help with the plumbing while using domain_driven.

## Installation

Add this line to your application's Gemfile:

    gem 'domain_driven_rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install domain_driven_rails
